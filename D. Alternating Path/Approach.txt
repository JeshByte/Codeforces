First we need to understand that as we can't add/remove edges, we need to handle different components seperately. Now if take a component and try to apply the given condition on it, we will realise that it is basically equivalent to turning it into a bipartite graph with the answer for that component being the max number of nodes among the 2 sets. So we will just apply 2 coloring algorithm/bfs on each unique component, and sum up all of their contributions to find the final total answer.
