Lets focus on the xor of 0 elements. Lets assume it is currently a1^a2^a5. Now we get the query 1 1 4. This effectively means we would need to remove a1 and a2 from that expression and add a3 and a4. Thus after this update it should become a3^a4^a5. This can be done by xorring a1^a2^a5 and a1^a2^a3^a4. Lets generalise this. We are given the query 1 l r. Then the new updated xor of 0 elements will be old_xor ^ xor of subarray from l to r. This same logic will also work for the xor of 1 elements. We can find the xor of the given subarrays fast using prefix xor. In this way we will be able to answer the queries in constant time.
