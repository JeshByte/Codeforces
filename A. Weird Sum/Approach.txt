Manhattan distance is basically the sum of row differences and column differences for all points. So we can compute them seperately. So lets say we find all the row coordinates of a certain color. Now we need to find an efficient method to find the row differences between all the possible pairs. As we need the absolute differences, knowing which number will be positive will be helpful. So we will sort it first. So for a point ri, its contri with all the points before it/less than it will be (ri-ri-1)+(ri-ri-2)+.... = i*ri - (ri-1 + ri-2 +...). Thus we can easily calculate each point's contri using a prefix running sum in linear time for the rows. We will find the same for columns similarly and add all of it up to get the final answer.
