So lets say we delete an array, and the cost comes out to be a1*a2+a3*a4*a6+a5. Now if we had taken only one element at a time(which satisfies the given condition too), the cost will be a1+a2+a3+a4+a5+a6. Basically the same expression but instead of *, we have +. Now is the sum of 2 elements always smaller than it's product? Yes, except for the case where one of the elements is 1. Thus what we can do is to first absorb all the ones into our first operation. We will be only able to do so with those 1s which have atleast one greater value after it. For that we just need to find the index of the first element from the right which is >1. Thus the final answer would be the sum of all the elements of the array, besides the 1s which are present before this above found index.
