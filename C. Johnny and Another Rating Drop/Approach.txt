The question can be framed as this -> how many times in total do the bits change in all of the positions in all numbers from 0 to n. And if we dry run a few test cases, we will observe that in the i-th position, the bits change every 2^i-th time. So for each position, the number of times the bits change are n/2^i(There are n+1 bits in each position but we ignore the last one as there is no bit after it for it to change into anything). So we just sum this up for all the positions and that will give us our final answer in logn time.
