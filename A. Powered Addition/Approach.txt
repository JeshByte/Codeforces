We can observe that the numbers that we add during each second can be seen as bits. So technically the total value that is added to an element throught out the total time can be anything if it's msb is <=total time. So we can just go through the array, and if an element is smaller than it's previous element we will just find the smallest value to add to it and then find it's msb. The highest msb out of all will give us our total time.
