So each element will be part of 3 types of subarrays - with elements to its left/right/both. So if we are able to satisy that it is more than enough. So we need to find what factors an element contributes to its gcd with its neighbours. We can find that by finding its gcd with its 2 neighbours. Now we can only reduce the number. So if we apply the operation we need to find the lowest number which will contribute all the factors for both of its neighbours which the original element was giving. Thus we need to basically find the lcm of the 2 gcds/contributions. If the lcm is <= the element, it is possible or else it isn't. In this way we will count the maximum number of operations that can be applied.
