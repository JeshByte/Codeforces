So lets say for some k, val is the answer. Then the differences in the indices of all the occurences of val in the array need to be <=k(We also need to consider its distance from the start and the end of the array, so for ease of computation also consider -1 and n as indices for all the values). We can simplify this by just keeping track of the max difference in index for each value. So if some val has max difference of diff, then it is an eligible answer for all k>=diff and k<=n. So for each max difference, keep track of the minimum eligible value. Now go through each k from 1 to n, and the answer for each k will be the minimum of the min value for each k starting from 1. For the values of k before you get one with a valid minimum answer, print -1 as there is no valid answer for them.
