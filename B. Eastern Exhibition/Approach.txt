So basically the average distance needs to be minimised. The denominator is constant so lets ignore that. We need to minimise the numerator then. The numerator is of the form |x1-x|+|x2-x|+....+|y1-y|+|y2-y|+.... Thus we can independently minimise the x and y terms. Now this is a standard idea that if we need to minimise |ai-x| where x has to be from the same range, x has to be the median of the given numbers. For odd length, it will be the middle most element. For even length, it will be the 2 middle most elements and the elements in between too. Thus if n is odd, the answer will be one. Otherwise, the answer will be the product of the number of such median terms for both the x and y coordinates.
