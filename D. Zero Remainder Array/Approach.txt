The least number we need to add to a[i] to make it divisible by k is req=Ceil(a[i],k)*k-a[i]. But the thing is when there are multiple numbers with the same such value we can set only one of them at the time. For the others we would need to set them to the next multiple of k, basically their req would get increased by k and so on. So basically if we divide all the numbers to different buckets as per their req value, the maximum x value needed to set all the values in the bucket would be req+(freq-1)*k where freq is the number of values in the bucket. So we would just find the maximum out of all these values and increment it by one(as x starts from 0) to get our final answer. We can use a hashmap for the bucket division.
