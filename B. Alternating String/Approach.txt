If we dry run a few test cases, we will realise that performing the given operation only helps with the ends of the segment it is being applied on. There is no effect on the rest of the segment. So a segment can have at most 2 bad ends which we can fix by using the given operation. What are bad ends in this case? Points where characters repeat. So we just need to count such points and if they are <=2, the answer is YES. Or else the answer is NO.
